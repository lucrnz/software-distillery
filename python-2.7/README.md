# Python 2.7

## Disclaimer
Even though this includes security patches, **avoid using Python 2.7 at all costs**, use it as a last resort, consider [PyPy 2.7](https://github.com/pypy/pypy/tags) instead.

## Description
Providing Python 2.7 binaries patched with latest [Ubuntu security updates](https://launchpad.net/ubuntu/+source/python2.7/2.7.18-13ubuntu1.5) + [OpenSSL v1.1 patched](https://github.com/kzalewski/openssl-1.1.1)

As tarballs & Docker containers for your easy convenience.
